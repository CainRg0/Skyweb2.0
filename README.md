# stankygae • links

Hub link stile “link-in-bio” con background animato, musica di sottofondo e interazioni audio.

## Demo
GitHub Pages: https://cainrg0.github.io/stankygae-links/

## Features
- Background animato su `anvas>`.
- Musica di background (`loser.mp3`) attivabile dal toggle.
- Click su avatar (`sky.jpg`) → riproduce `gatito.mp3`, interrompe la musica e poi la ripristina a fine audio.
- Tasto **OF ♥** → apre popup (`gatto.jpg`), riproduce `troll.mp3`, si chiude automaticamente a fine audio e ripristina la musica.
